# מחשבון כדאיות: עוסק מורשה ↔ חברה בע״מ | יער צפון

מחשבון אינטראקטיבי להשוואת כדאיות מס בין עוסק מורשה לחברה בע״מ, מבוסס נתוני 2026.
קובץ HTML עצמאי אחד (`index.html`) — כל הלוגיקה והלוגו מוטמעים בפנים, בלי תלות חיצונית.

## פרסום (Netlify + GitHub)
1. הקובץ `index.html` יושב בשורש ה-repo.
2. Netlify מחובר ל-repo ומפרסם אוטומטית בכל שינוי (commit).
3. **לעדכון:** ערוך את `index.html` ב-GitHub (עיפרון → Commit) או העלה גרסה חדשה (Add file → Upload files). Netlify יפרסם תוך כדקה.

## לפני פרסום ראשון — למלא בראש הקובץ (בלוק `BRAND`)
- `websiteUrl` — כתובת האתר
- `facebookUrl` — קישור הפייסבוק
- `whatsappPhone` — מספר וואטסאפ בפורמט בינלאומי (972 ואז המספר בלי 0)

## בדיקה שנתית של הפרמטרים (פאנל "🧮 שיעורי מס" בתוך המחשבון)
כל הערכים נערכים בממשק, אבל מומלץ לאמת שנתית מול המקורות הרשמיים:
- מדרגות מס הכנסה ומס יסף
- מס חברות (23%) ומס דיבידנד (30%)
- ביטוח לאומי + בריאות לעצמאי (משולב 6.92%/15.79%, בריאות 3.23%/5.17%) + ניכוי 52%
- ביטוח לאומי לבעל שליטה שכיר (עובד 4.25%/11.96%, מעסיק 4.46%/7.38%)
- תקרות פנסיה/קרן השתלמות לעצמאי (הכנסה מזכה ₪232,800 / ₪293,397)
- טבלת זיכוי יישוב מזכה (489 יישובים, מוטמעת) — מתעדכנת בלוח הניכויים של רשות המסים
- שווי נקודת זיכוי (₪242/חודש), עלויות תפעול ואגרות
